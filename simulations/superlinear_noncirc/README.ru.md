# `superlinear_noncirc` — нециркулярный тест флора C3 (ностальгия выходит на плато ν_C1 < 1)

Статья #3 (`landauer-undertow-oa`), спутник / починка `superlinear_memory`
(supplementary § S4.4 / § S4.5). Определения ν, ν^theor, ν_C1, ε_track, Σ_∞ —
по [Andriishin2026b].

## Что тестируется

Для обучателя со **сверхлинейно** растущей ёмкостью памяти `|M(t)| = M₀·t^α`
(`α ∈ {1, 1.5, 2, 3}`): выходит ли *oracle-нормированная ностальгия* ν^theor на
**плато у флора слежения C1** `ν_C1 < 1` — а не уползает к 1?

Существующая симуляция `superlinear_memory` оценивает каждый удержанный снимок
побитовым порогом «полезен, если возраст < τ_E», поэтому её табулируемый
per-bit-мажорант `ν_naive = 1 − φ(t) → 1`. Сама статья помечает это как
**циркулярность** (зашивает побитовую равномерность) и утверждает, что истинная
ν^theor удерживается у флора C1. Этот спутник доказывает это напрямую: он считает
ν^theor из **реального совместного гауссова апостериора**, **без** побитового
порога по возрасту, и показывает выход на плато у аналитического
`ν_C1 = Σ_∞/σ_0²`.

## Модель (мультипликативная — θ *есть* коэффициент перехода, не эмиссия)

`K = 56` независимых скалярных координат. На координату:

```
X_{t+1} = θ_t · X_t + w_t ,        w_t ~ N(0, q)        (состояние)
θ_{t+1} = (1−λ) · θ_t + u_t ,       u_t ~ N(0, σ²)       (OU-дрейф коэффициента)
```

θ наблюдается **только мультипликативно**, через пару состояний `(X_t, X_{t+1})`:
при известном `X_t` пара — гауссово наблюдение `θ_t` с планом `X_t` и шумом `q`,
т. е. пошаговая информация Фишера `X_t²/q`. Скорость Фишера
`I_rate = E[X_t²]/q` **измеряется в прогоне** (≈ 1 при малых θ, не постулируется).
Параметры: `λ = 0.1`, `σ² = 0.002`, `q = 1` ⇒ адиабатика `σ_0² = σ²/(2λ) = 0.01`,
`ε_track = I_rate·σ²/λ² ≈ 0.20`. `|θ| ≪ 1` (адиабатические хвосты пренебрежимы),
так что линейно-гауссов **фильтр Калмана — точный оптимальный обучатель**.
`T = 10⁵`, `R = 48` реализаций, `M₀ = 100`, `τ_E = 1/λ = 10`, seed `20260527`,
только numpy.

## Нециркулярный обучатель (суть)

Растущий FIFO удерживает `M(t) = M₀·t^α` самых свежих пар. Из них обучатель
строит **реальный совместный гауссов апостериор для текущего θ_t** — рекурсивное
обновление Калмана/RLS, где каждая пара — гауссово правдоподобие по θ с точностью
`X_s²/q`, а более старые удержанные пары понижаются в весе OU-забыванием (шаг
предсказания раздувает дисперсию на `σ²` каждый шаг, `τ_f ≈ τ_E = 1/λ`). Несколько
**свежих** снимков **одной и той же** координаты *совместно* стягивают оценку к
`Σ_∞` — вытекший из горизонта прошлый θ неинформативен о текущем θ_t, так что
накопление большего объёма (устаревшей) памяти **не** опускает Σ ниже `Σ_∞`.
Побитового порога «полезен, если возраст < τ_E» **нет**.

Поскольку `M₀·t^α ≥ t` при любом `α ≥ 1` (и `M₀ ≥ 1`), FIFO удерживает **всю
историю** для всех α, так что совместный апостериор для θ_t совпадает с
полнопамятным калмановским для каждого α — значит **ν^theor в точности не зависит
от α**. Флор задаётся недавним окном ~`τ_conc` (см. `fig_noncirc_window_floor.png`);
удержание большего (что все эти α и делают) не опускает Σ ниже `Σ_∞`.

## Аналитическая мишень (воспроизводится численно, независимая сверка)

```
σ_0²      = σ²/(2λ)                                    (дисперсия OU-латента)
ε_track   = I_rate·σ²/λ²
Σ_∞       = (λ/I_rate)(√(1+ε_track) − 1)               (флор слежения Риккати)
ν_C1      = Σ_∞/σ_0² = (2/ε_track)(√(1+ε_track) − 1) ≈ 1 − ε_track/4  (≈ 0.954)
```

`ν_C1 < 1` — **флор ностальгии**: оптимальный обучатель навсегда захватывает
остаточную долю `1 − ν_C1 ≈ ε_track/4` предсказательной информации оракула — это
ненулевой флор, а не полное устаревание.

## Три нециркулярные меры ностальгии (θ известно по построению)

При `E[D_KL] = E[(θ_t − θ̂_t)²·X_t²]/(2q)` и предсказанной оценке θ̂_t (обучатель
обязан предсказать `X_{t+1}` до его наблюдения):

| мера | определение | роль |
|---|---|---|
| (i) `ν_varratio` **ОСНОВНАЯ** | `E[(θ−θ̂)²]/E[θ²]` | ведущий Fisher-сокращающийся вид = `Σ_∞/σ_0²` статьи (§S1.3); малошумная, плоская |
| (ii) `ν_predinfo` (буквальная) | `E[D_KL] / (½ E[−ln(1−θ²)])` | X²-взвешенная KL над взаимной информацией пары (явная формула задания) |
| (iii) `ν_fullwtd` | `E[D_KL] / (E[θ²X²]/2q)` | полностью X²-взвешенная; ~1.6% ниже ν_C1 из-за субведущей антикорреляции план–ошибка, которую замкнутая форма отбрасывает |

В разложении KL (§S1.3) Фишер `I` сокращается (`½IΣ / ½Iσ_0² = Σ/σ_0²`), поэтому
**variance-ratio** — верный измеряемый аналог `ν_C1`; две X²-взвешенные —
уточнения. Все три **выходят на плато** и **окаймляют ν_C1 в пределах ~1.6%**.

## Результат (фикс. seed 20260527)

Измерено (плато на `t ≥ 10⁴`): `I_rate = 1.022`, `ε_track = 0.204`,
`ν_C1(cont) = 0.9535` (дискретные скобки 0.9485 / 0.9583).

| мера ν^theor | плато | к ν_C1(cont) |
|---|---:|---:|
| (i) variance-ratio (основная) | **0.9628** | +0.97 % |
| (ii) predictive-info (буквальная) | 0.9700 | +1.73 % |
| (iii) полностью X²-взвешенная | 0.9465 | −0.74 % |

- **(1) Плато совпадает с ν_C1: PASS.** Основная ν^theor плоская (наклон
  `+1.2·10⁻³` на ln t; половины плато 0.9619 / 0.9637) на уровне
  **0.963 ≈ ν_C1 = 0.954** (<1 %), **не зависит от α** (FIFO = вся история для
  всех α). Три меры окаймляют ν_C1 в пределах ~1.6 %.
- **(2) Σ̂ → Σ_∞ (флор, не 0): PASS.** Σ̂ = 0.01010, реализованная MSE = 0.01015 ≈
  корень дискретного Риккати `Σ_∞ = 0.01009` — обучатель достигает **флора**, а не
  нуля.
- **(3) Предельные контроли: PASS.** Без памяти (θ̂ = 0) ⇒ ν = 1.000 (точно);
  один снимок ⇒ ν = 0.992 (~1) — воспроизводит артефакт. Оракул (θ̂ = θ) ⇒
  ν = 0.000 (точно).
- **(4) Наивный мажорант расходится: PASS.** `ν_naive = 1 − φ → 1` для каждого α
  (0.9997–0.9999 при T), поднимаясь **выше** плато 0.963 — заглавная расходимость,
  подтверждающая, что старый пороговый `→ 1` есть циркулярный артефакт. При этом
  `η_v = I_pred/I_mem → 0` (`I_mem ∝ M₀·t^α`) со скоростью, зависящей от α.
- **(5) I_pred^opt > 0: PASS** (5.5·10⁻³, невырожденно).

**Вердикт.** Ностальгия ν^theor при растущей памяти **выходит на плато у флора C1
ν_C1 < 1** и **не зависит от α**, совпадая с аналитическим `Σ_∞/σ_0²`;
per-bit-мажорант `ν_naive → 1` — циркулярный артефакт. `η_v = I_pred/I_mem → 0`
несёт фактическое закрытие маршрута (двухтрековая история). Обучатель достигает
`Σ_∞`, а не 0.

### Честные пометки

- Три меры ностальгии разбросаны на ~1.6 % вокруг ν_C1. Это **субведущая
  корреляция план–ошибка** (адаптивный Калман слежит точнее именно когда `X_t²`
  велико, так что X²-взвешивание сдвигает захваченную долю); замкнутая форма
  `ν_C1 = Σ_∞/σ_0²` её отбрасывает. Variance-ratio — та, которой соответствует
  вывод статьи — совпадает <1 %.
- Малый зазор непрерывное-против-дискретного (`ν_C1` cont 0.9535 против
  discrete-predicted 0.9583) — это `O(λ)` временна́я дискретизация OU-дрейфа;
  дискретный корень Риккати приводится рядом с непрерывным заголовочным.
- Численный шаг `τ = 1` (рекомендуемый) везде; фильтр — точный линейно-гауссов
  оптимум, поэтому в ν^theor не входит приближение оценщика сверх Монте-Карло-шума
  (θ известно по построению, так что `ν^theor = ν^op`).
