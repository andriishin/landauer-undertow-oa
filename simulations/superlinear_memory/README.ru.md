# `superlinear_memory` — фазовая граница (Теорема 3, C3)

Статья #3 (`landauer-undertow-oa`) § 5 / § 6 пункт 4 / supplementary § S4.4.

## Что вычисляется

Ёмкость памяти растёт как `|M(t)| = M0·t^α` для `α ∈ {1, 1.5, 2, 3}`
(полиномиальный рост) и, отдельно, `|M(t)| = M0·e^{κt}` (экспоненциальный
побег). FIFO-обновление. Управляющая величина — **доля свежих битов**

```
φ(t) = (|M(t)| − |M(t−τ_E)|) / |M(t)| = 1 − ((t−τ_E)/t)^α  ≈  α·τ_E/t
```

Мы фитируем `φ(t) ~ t^{−γ}` на хвосте `t ≥ 2000` и сообщаем:

- показатель затухания `γ` для каждого полиномиального `α` (утверждение:
  `γ = 1` для **любого** конечного `α`, так что доля свежих `φ(t) → 0`, откуда
  **нет полиномиального `α_c`**). Табулируемая `ν(T) = 1 − φ` — это *наивная
  арифметическая per-bit* доля: она → 1, но это **циркулярная** величина
  пороговой модели (зашивает побитовую равномерность) и **не** есть `ν^theor`.
  Несущее закрытие маршрута — `η_v = I_pred/I_mem → 0`; истинная `ν^theor`
  удерживается у флора слежения C1 (`ν^theor → ν_{C1} < 1`, не к 1);
- префактор `φ·t → α·τ_E`;
- для экспоненциального роста — постоянный предел доли свежих
  `φ_∞ = 1 − e^{−κτ_E} > 0` (побег, `ν^theor < 1`) и отношение ёмкостей `|M(T)|/|M(0)|` —
  при этом горизонт-ограниченный числитель `I_pred ≤ H_τ` над растущим знаменателем `I_mem`
  уводит `η_v = I_pred/I_mem → 0` структурно, по *информационной*, а не энергобюджетной, причине.
  (Истинная `I_mem ≤ H(X_E^{≤t})` растёт лишь ~линейно; в прогоне взята `I_mem ∝ |M|` как
  консервативная верхняя оценка, поэтому η_v-отношение = `|M(0)|/|M(T)|` завышает спад — фактический ~1/t.)

## Параметры

- `k = 8`, `K = 56`, `τ_E = 200`, `M0 = 100`
- полиномиальный `α ∈ {1, 1.5, 2, 3}`; экспоненциальный `κτ_E ∈ {0.1, 0.5, 1.0}`
- `T = 20000`, окно фита `t ≥ 2000`, 400 log-spaced моментов измерения
- `SEED = 20260529` (seed статьи #2 + 5) — **декоративный / не используется**:
  выход представляет собой детерминированную табуляцию замкнутой формы
  `φ = 1 − ((t−τ_E)/t)^α` — **никакого Монте-Карло нет**, `np.random` ни разу
  не вызывается, RNG ни разу не сидируется. Значение сохранено только для
  провенанса.

## Фактический результат (фиксированный seed)

Полиномиальный рост:

| α   | γ (φ~t^−γ) | expected | φ·t   | α·τ_E | ν(T)   |
|----:|-----------:|---------:|------:|------:|-------:|
| 1.0 | 1.0000     | 1.000    | 200.0 | 200   | 0.9900 |
| 1.5 | 0.9908     | 1.000    | 297.6 | 300   | 0.9850 |
| 2.0 | 0.9816     | 1.000    | 393.7 | 400   | 0.9801 |
| 3.0 | 0.9636     | 1.000    | 581.3 | 600   | 0.9703 |

Экспоненциальный побег:

| κτ_E | φ_∞    | ν_∞    | \|M(T)\|/\|M(0)\| |
|-----:|-------:|-------:|--------------------:|
| 0.1  | 0.0952 | 0.9048 | 1.99e+04            |
| 0.5  | 0.3935 | 0.6065 | 3.14e+21            |
| 1.0  | 0.6321 | 0.3679 | 9.84e+42            |

- **γ = 1 независимо от α: ПОДТВЕРЖДЕНО.** γ ∈ {1.000, 0.991, 0.982, 0.964},
  разброс 0.036, все ≈ 1. Показатель затухания **не** зависит от α, так что
  доля свежих `φ(t) → 0` для любого конечного полиномиального α — **нет
  полиномиального порога побега `α_c`**, в согласии с Теоремой 3.
  (Табулируемая арифметическая `ν(T) = 1 − φ → 1` — наивная/циркулярная per-bit
  мажоранта, **не** `ν^theor`: `ν^theor` в статье удерживается у флора C1
  `→ ν_{C1} < 1`, не к 1; несущее закрытие маршрута — `η_v → 0`.)
  Видимый тренд γ вниз до 0.964 при больших α на базовом окне
  (`t ≥ 2000`) — это **конечно-оконный subleading-эффект, а не зависимость от
  α**: по мере сдвига окна фита вправо γ → 1 для **всех** α (см. блок *gamma
  window-convergence check* в `results_summary.{txt,json}`).
- **Префактор φ·t → α·τ_E: ПОДТВЕРЖДЕНО** (200/298/394/581 против 200/300/400/600).
- **Экспоненциальный побег: ПОДТВЕРЖДЕНО.** `φ_∞ > 0`, так что `ν^theor`
  удерживается ниже 1, но ёмкость `|M| ∝ e^{κt}` расходится (отношения вплоть до
  9.8e42 за прогон) при неподвижном `I_pred ≤ H_τ`, так что `η_v = I_pred/I_mem → 0` структурно —
  побег от пола ностальгии не помогает, по *информационной* (не энергобюджетной) причине
  (Теорема 3 § 5.4). Истинная `I_mem ≤ H(X_E^{≤t})` растёт ~линейно, фактический спад η_v ~1/t;
  отношение ёмкостей — консервативная верхняя оценка.

### Честная оговорка — теперь проверяемая в симуляции

Малый тренд γ вниз от 1.000 (α=1) до 0.964 (α=3) — это **артефакт конечного
окна фита**: точная `φ = 1 − ((t−τ_E)/t)^α` имеет subleading-кривизну,
уплощающую лог-лог-наклон ниже 1 для больших α при конечном t; ведущий
асимптотический показатель тождественно равен 1 для всех α (проверка
префактора `φ·t → α·τ_E` подтверждает ведущий порядок `t^{−1}`). Это не влияет
на качественный вывод (γ → 1, нет α_c).

Это больше не просто постулируется — `gamma_window_convergence()` **доказывает**
это, перефитируя γ на однодекадных окнах `[w, 10w]` точной φ с растущей нижней
границей `w` (`w=2000` воспроизводит базовый фит):

| fit window | α=1.0 | α=1.5 | α=2.0 | α=3.0 |
|-----------:|------:|------:|------:|------:|
| t ≥ 2e3    | 1.000 | 0.991 | 0.982 | 0.963 |
| t ≥ 2e4    | 1.000 | 0.999 | 0.998 | 0.996 |
| t ≥ 2e5    | 1.000 | 1.000 | 1.000 | 1.000 |
| t ≥ 2e6    | 1.000 | 1.000 | 1.000 | 1.000 |

γ → 1 **монотонно для каждого α** по мере сдвига окна вправо. Локальный
лог-лог-наклон точной φ при α=3 идёт `−0.897` (t=2e3) → `−0.990` → `−0.999` →
`−1.0000` (t ≥ 2e7): показатель ведущего члена тождественно равен 1, так что
тренд строго subleading и **нет α_c**.
