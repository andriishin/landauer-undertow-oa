# `fou_floor` — пол ностальгии и показатель скорости сноса для ТОЧНОГО стационарного fOU (Теорема 2, C2)

Статья #3 (`landauer-undertow-oa`) § 6 пункт 2 / supplementary § S4.2.

## Почему точный стационарный fOU (а не суррогат)

Наивная **AR(1)-свёртка дробного гауссова шума** (`θ ← α·θ + ξ`) —
«langevin-filtered fGn surrogate» / «AR(1)∘fGn surrogate» — это **не**
стационарный дробный процесс Орнштейна–Уленбека (fOU): его низкочастотный спектр
задаётся полкой AR(1), а не нормировкой Ланжевена–fOU, поэтому её эмпирический
показатель хвоста перемешивания не равен аналитическому `4(1−H)` (он `2.23` при
`H=0.7` и `1.26` при `H=0.9`, оба далеки от `1.2` / `0.4`, причём порядок
относительно аналитики искажён). Это расхождение — **артефакт неправильного
процесса**: чистый тест требует **прямой генерации стационарного fOU**.

**Здесь латент генерируется как *точный* стационарный fOU**, спектрально, и
аналитический показатель `4(1−H)` **на нём восстанавливается** (в дальнем хвосте).
Расхождение — свойство суррогата AR(1)∘fGn, не теории.

## Метод (точный стационарный fOU, только numpy.fft — без scipy)

Стационарный fOU `dY = −λY dt + σ dBᴴ` имеет спектральную плотность

```
S(ω) ∝ |ω|^{1−2H} / (λ² + ω²)
```

(приращения fBm дают `|ω|^{1−2H}`, OU-фильтр `1/(λ+iω)` даёт
`1/(λ²+ω²)`).

- **Циркулянтные собственные значения = дискретно-временной PSD на циркулянтной сетке**
  `ω_j = 2π j / M` (Винер–Хинчин). Мы строим их **напрямую** как *свёрнутый*
  дискретный PSD (сумма `S` по алиасам на `(−π, π]`; интегрируемая `ω=0`
  сингулярность для `H>½` усредняется по бину). Построенные так собственные значения
  **неотрицательны по построению** → вложение положительно определено с
  **нулевым отсечением** (прогон логирует `min_eig` и число отрицательных и подтверждает
  `min_eig > 0`, `n_neg = 0` для каждого `H`).
- **Синтез Davies–Harte** из этих собственных значений (`numpy.fft`) даёт гауссовы
  стационарные траектории ровно с этой ковариацией.
- **Целевая автоковариация** `γ(k)` = обратное DFT собственных значений — *тот же*
  массив, из которого генерируются траектории — точна вплоть до лага `M/2` (за которым
  циркулянт заворачивается). Показатель дальнего хвоста считывается только до лага `M/4`.

### Контроль ковариации

Эмпирическая автокорреляция реализованных траекторий `ρ̂(s)` сравнивается с целевой
`γ(s)`: они согласуются там, где конечно-выборочный оценщик ACF несмещён
(короткие/умеренные лаги — отн. ошибка `< 5 %` вплоть до лага ~200–400). На **очень больших
лагах оценщик sample-ACF сам по себе сильно смещён к 0** (свойство
оценщика на данных с длинной памятью, *а не* траекторий), поэтому **показатель
дальнего хвоста считывается из точного `γ`**, который траектории воспроизводят по построению,
а не из реализованного ACF.

### Прямой тест показателя

Из точного `γ` мы формируем `ρ²(s) = 2β(s)` и измеряем показатель его степенного
хвоста в **дальних окнах** `Δ ∈ [8, 40] τ_E` и `[20, 80] τ_E` (где
степенной хвост проявился за OU-углом), сравнивая с `4(1−H)`. Мы
**также** приводим показатель эмпирического ACF в *исходном* доступном окне
`[τ_E, 8 τ_E]` с bootstrap CI по реализациям, чтобы задокументировать точно, что
именно разрешает конечно-выборочный оценщик. **Без фита к цели.**

## Модель ностальгии / считывание пола

FIFO-память из `|M| = 400` снимков-битов отслеживает реализованные `K = k(k−1)` fOU-
координаты. Удержанный бит возраста `Δ` считается предиктивным, пока его остаточная
предиктивная ценность `ρ²(Δ)` превышает DPI-горизонт `ρ²(τ_E)`; иначе он ностальгический.
`ν(t)` = ностальгическая доля. `c ≈ 0.30` (`refresh_period = τ_E / c`), так что
пол теории — `1 − c = 0.70`.

## Фактический результат (фиксированный seed, точный стационарный fOU)

**Вложение (циркулянт свёрнутого-PSD): положительно определено по построению, нулевое
отсечение** — `min_eig = 0.43 / 0.24 / 0.14 / 0.086` и `n_neg = 0` для
`H = 0.3 / 0.5 / 0.7 / 0.9`. Инфраструктурный исход (c) *не* возникает.

### Пол (несущее утверждение C2) — инвариантен

| H   | floor (last 25%) | liminf |
|----:|-----------------:|-------:|
| 0.3 | 0.7000           | 0.7000 |
| 0.5 | 0.7000           | 0.7000 |
| 0.7 | 0.7000           | 0.7000 |
| 0.9 | 0.7000           | 0.7000 |

`liminf ν = 0.7000 ≈ 1 − c` для каждого `H`, разброс `0.0000`. Константа пола

`ρ²(τ_E)`; симуляция *иллюстрирует* инвариантность на корректных траекториях — само
доказательство в § S2.)

### Показатель скорости сноса `4(1−H)` (`H > ½`) — из ТОЧНОЙ fOU-ковариации

| H   | analytic `4(1−H)` | far `[8,40]τ_E` | far `[20,80]τ_E` | near `[1,8]τ_E` | emp-ACF `[1,8]τ_E` (95% CI) |
|----:|------------------:|----------------:|-----------------:|----------------:|----------------------------:|
| 0.7 | 1.200             | **1.237**       | **1.201**        | 1.624           | 2.436 [2.36, 2.52]          |
| 0.9 | 0.400             | **0.406**       | **0.385**        | 0.458           | 1.398 [1.37, 1.43]          |

(`H = 0.3, 0.5` дают экспоненциальные хвосты; темп эмпирического ACF для `H = 0.5` —

### Исход (a): `4(1−H)` ВОССТАНАВЛИВАЕТСЯ на корректном стационарном fOU

- В **дальнем окне** (`Δ ≥ 8 τ_E`), за OU-углом, локальный log-log
  наклон `ρ²` близко соответствует `4(1−H)`: `H=0.7 → 1.24` (`[8,40]τ_E`) и `1.20`
  (`[20,80]τ_E`) против аналитического `1.20`; `H=0.9 → 0.41` и `0.39` против аналитического
  `0.40`. Правая панель `fig_fou_approach.png` показывает, что обе точки ложатся на
  аналитическую линию `4(1−H)`.
- Он **не изолирован в исходном окне `[τ_E, 8 τ_E]`** (OU-угол
  всё ещё загрязняет там: `1.62` / `0.46`), как и **конечно-выборочным
  оценщиком sample-ACF** в этом окне (`2.44` / `1.40`) — sample ACF
  смещён к 0 на больших лагах, где живёт степенной хвост, поэтому он не может
  разрешить дальний хвост на доступных длинах траекторий.

Итак, `4(1−H)` — это **асимптотический показатель дальнего хвоста истинного fOU**:
восстанавливается на корректном процессе, но только в дальнем окне и из ковариации
процесса (которую траектории воплощают по построению), а не из sample ACF в ближнем
окне. Это **апгрейдит C2**: **глубина `1 − c` доказана** (Теорема 2), а
**показатель `4(1−H)` теперь подтверждён на корректном процессе**. Расхождение —
свойство **суррогата** AR(1)∘fGn, не теории.

### Честные оговорки

- Показатель дальнего хвоста считывается из **точной вложенной ковариации `γ`**, а не
  из реализованного sample ACF: оценщик sample-ACF на данных с длинной памятью
  сильно смещён к 0 на больших лагах, поэтому он не может изолировать степенной хвост
  на доступной длине траектории. Используемое `γ` — это ровно та ковариация, из которой
  генерируются траектории (это обратное DFT тех самых циркулянтных собственных значений, что
  использует синтез Davies–Harte), так что это свойство *процесса*, проверенное
  против реализованных траекторий там, где оценщик надёжен (контроль ковариации,
  короткие/умеренные лаги) — а не пере-фит замкнутой формы.
- Показатель всё ещё **чувствителен к окну** и **асимптотичен**: в ближнем
  окне `[τ_E, 8 τ_E]` OU-угол толкает его выше `4(1−H)` (`1.62` при
  `H=0.7`); он сходится к `4(1−H)` только когда `Δ ≳ 8 τ_E`. Это отражает
  конечно-оконную транзиентность C1 (§ 7): аналитическое значение асимптотично и не
  изолировано на наименьшем окне — но, что принципиально, оно *изолировано* на
  корректном процессе в достаточно дальнем окне, что было невозможно с суррогатом AR(1)∘fGn
  на любом окне.
- `H = 0.9` лежит в **диапазоне длинной памяти `H ≥ 3/4`, который строгая Теорема 2
  не покрывает** (`∫β ds` расходится, § S2.2); пол там — численно
  поддержанная гипотеза только, и показатель сходится медленнее (значение
  `[20,80]τ_E` `0.385` слегка ниже `0.40`).
